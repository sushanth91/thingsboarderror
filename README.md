## Run Locally

Clone the project

```bash
  git clone https://github.com/thingsboard/thingsboard.git
```

Go to the project directory

```bash
 cd D:\thingsboard
```

Install dependencies

```bash
npm install -g cross-env 

npm install -g npm@10.5.0

npm install -g webpack 

```
